## 🤖 Android Updates

### 🔴 Android Security Bulletin (August 2025)
<div style="border-left: 4px solid #ef4444; padding-left: 1rem; margin: 1rem 0;">
  <div style="display: flex; align-items: center; gap: 0.5rem; margin-bottom: 0.5rem;">
    <span style="background: #ef4444; color: white; padding: 0.25rem 0.5rem; border-radius: 4px; font-size: 0.875rem;">CRITICAL P0</span>
    <span style="color: #6b7280;">SPL 2025-08-01/05</span>
  </div>
  
  **Why it matters:** System RCE vulnerability (Critical) and framework EoP. Pixel devices patched at 2025-08-05 level.
  
  **Impact Areas:**
  - 🔐 Login flows
  - 📱 OTP verification  
  - 💳 3DS authentication
  
  **Required Actions:**
  1. ✅ Update test devices to SPL 2025-08-05
  2. ✅ Run smoke tests (login, transfers, 3DS)
  3. ✅ Monitor crash/error rates for 48h post-update
  
  **Test Matrix:** Android API 29-35 / WebView 139.x
</div>

### 🟠 Google Play API 35 Requirement
<div style="border-left: 4px solid #f97316; padding-left: 1rem; margin: 1rem 0;">
  <div style="display: flex; align-items: center; gap: 0.5rem; margin-bottom: 0.5rem;">
    <span style="background: #f97316; color: white; padding: 0.25rem 0.5rem; border-radius: 4px; font-size: 0.875rem;">HIGH P1</span>
    <span style="color: #6b7280;">Deadline: Aug 31, 2025</span>
    <span style="background: #10b981; color: white; padding: 0.25rem 0.5rem; border-radius: 4px; font-size: 0.875rem;">Extension: Nov 1</span>
  </div>
  
  **Why it matters:** Apps failing to meet requirement will be blocked from submission/updates.
  
  **Impact Areas:**
  - 📱 Store submission
  - 🏗️ Build/Release pipeline
  
  **Required Actions:**
  1. ⚠️ Set `targetSdkVersion=35`
  2. ⚠️ Update blocked dependencies (permissions/file access)
  3. ⚠️ Run Android 15 compatibility tests
     - Predictive back gesture
     - Behavior changes
  
  **Test Matrix:** Android 15 (API 35)
</div>
